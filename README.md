# Transfer Equity Distillation

This repository contains the official implementation of our paper:

**Do MLPs Inherit GNN Knowledge Uniformly? Class-Conditional Transfer Equity in Graph-Free Distillation**

We propose **TED (Transfer Equity Distillation)**, a training-time objective augmentation for graph-free GNN-to-MLP distillation. TED measures whether correct teacher knowledge is inherited uniformly across classes and reduces class-conditional missed transfer while preserving the deployed student as a graph-free MLP.

Code, training scripts, and experiment configurations will be released here.
